04_부트스트랩_이미지 관련 서식



1. rounded : 모서리 라운드



2. rounded-circle : 모서리 둥글게 _ 원모양주기



3. img-thumbnail : 이미지 섬네일 처럼 보이게 하기

.img-thumbnail : border, padding, borderradius 속성값 사용



4. 이미지와 문자를 사용하여 콘텐츠 제작 figcaption, figure

class="img-fluid" : 이미지를 반응형으로 동작하게 함(적용시 width:100%, height: auto속성이 자동으로 적용됨)

부모의 크기가 변하면 이미지 크기도 같이 반응함
