02_부트스트랩_글자, 문단

부트스트랩에서는 기본 글자크기가 16px이고, 행높이(줄간격)는 1.428이 기본값이다.

위 내용은 문단태그(p)요소에 자동으로 적용되며, 하단여백(margin)은 16px이 기본값이다.

부트스트랩 h1 - 35.95px, calc(1.375rem + 1.5vw)

부트스트랩 h2 - 29.57px, calc(1.375rem + 1.5vw)

부트스트랩 h3 - 26.38px, calc(1.3rem + .6vw)

부트스트랩 h4 - 23.19px, calc(1.275rem + .3vw)

부트스트랩 h5 - 20px, 1.25rem
부트스트랩 h6 - 16px, 1rem

제목요소 h태그는 margin-bottom 8px이 기본값이다.

1. 텍스트 관련 서식

small : 글자를 작게 설정 _ 14px

mark : 글자에 형광펜을 친 것 같은, 하이라이트 효과를 적용함 (부트스트랩이 아닐 경우 노랑배경으로 나타남)

abbr : 약어나 줄임말을 사용하고자 할 때 사용함

blockquote

blockquote : 다른 사람의 말을 대신하여 전하는 글(글자크기:14px, 색상:#6c757d)

용어제목_굵게
용어설명_기본

code : 문서안에서 태그이름을 표시하기 위한 태그로서 색상이 '자주색'으로 변경됨.
 let gnb = document.getQuerySelector('.gnb');

  • text-muted : 텍스트의 중요도에 따라 문자의 색상을 다르게 표시할 수 있음
  • text-primary : 텍스트의 중요도에 따라 문자의 색상을 다르게 표시할 수 있음
  • text-success : 텍스트의 중요도에 따라 문자의 색상을 다르게 표시할 수 있음
  • text-info : 텍스트의 중요도에 따라 문자의 색상을 다르게 표시할 수 있음
  • text-wraning : 텍스트의 중요도에 따라 문자의 색상을 다르게 표시할 수 있음
  • text-danger : 텍스트의 중요도에 따라 문자의 색상을 다르게 표시할 수 있음

상황별로 문자의 배경색을 적용하려면
class="bg-primary", class="bg-success", class="bg-info", class="bg-wraning", class="bg-danger"
을 p요소안에 작성한다.

2. 문단 관련 서식

class="lead" : 단락을 눈에 띄게 하기(글자크기 : 20px)

class="small" : 상위문자 크기의 85%=14px로 문자 크기를 작게 설정함

class="text-start" : 문단 정렬 _ 왼쪽 정렬

class="text-end" : 문단 정렬 _ 오른쪽 정렬

class="text-center" : 문단 정렬 _ 중앙 정렬

class="text-justify" : 문단 정렬 _ 양쪽 배분 정렬

class="text-justify" : 문단끝에서 글자가 다음줄로 넘어가지 않게 하기

class="text-lowercase" : 문자를 소문자로 출력 ABC

class="text-uppercase" : 문자를 대문자로 출력 abc

class="text-capitalize" : 영문자의 첫글자만 대문자로 출력 abc

목록태그에 class="list-unstlye : 목록태그의 기호 숨기기(css -> list-stlye:none;)

목록태그에 class="list-inline-item" : 블럭요소를 가로배치하기 위함
