반응형 레이아웃

  • 가변단위 : %, vw, vh, rem, em
  • viewport속성 : 디바이스별 해상도에 맞게 가로 크기 맞춰줌
  • 미디어쿼리

부트스트랩에서 그리드 레이아웃 속성을 사용하여 반응형 그리드 레이아웃을 사용할 수 있음

부트스트랩에서는 하나의 행에 반드시 12개의 열(컬럼)을 사용하여야 함

만약 12개의 열(컬럼)을 모두 사용하지 않으려면, 더 넓은 열(컬럼)을 만ㄷ르기 위해 열(컬럼)들을 하나의 그룹으로 만들어 사용할 수 있다.

그리그 시스템에서 사용할 수 있는 디바이스별 클래스
